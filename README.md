\# 🧠 SmartGrieve — AI-Based Grievance Redressal System



An AI-powered civic complaint management platform that automatically

classifies, prioritizes, and routes citizen complaints to the correct

government department.



\## 🚀 Features

\- Auto classify complaints

\- Predict complaint priority

\- Detect duplicate complaints

\- Route to correct department

\- Admin dashboard



\## 🛠️ Tech Stack

\- Frontend: React

\- Backend: Flask/Python

\- Database: MongoDB

\- AI/ML: Scikit-learn
